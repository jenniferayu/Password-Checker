# Password Checker (C++)

## Description

This program checks whether a password is strong or weak based on basic security rules. It was created as a lab project using C++.

## Features

* Checks password length
* Detects numbers in the password
* Detects special characters
* Provides feedback on password strength

## Technologies Used

* C++

## How to Compile

Use a C++ compiler such as g++.

Example:
g++ password_checker.cpp -o password_checker

## How to Run

After compiling, run the program:

./password_checker

## Example

Input:
Enter a password: Hello123!

Output:
Strong password
